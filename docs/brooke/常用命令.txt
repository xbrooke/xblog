git提交命令
打开git命令行

鼠标右键-->git bash here

首先登录：

git config user.name "你的名字"

git config user.email "你的邮箱"

登陆后在命令窗口内逐次输入以下内容

1、git pull下拉代码到本地



2、git status查看工作树



3、git add . 添加文件



4、git commit -m "提交备注 ref 任务号"



5、git push推送到服务器上

提示：必须有权限，否则是无法提交成功的
